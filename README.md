# html-manager
JavaScript + LAMP solutions to HTML CMS management.

# Introduction
This repo is to build a HTML management portal for projects that require HTML/CSS/Bootstrap/JS or any other frameworks.
And managed by a single LAMP built services that able to use the Web Standards together.

# Goals
1. To build a standard system to manage all HTML repos.
2. To follow the web standards so that the system can be sustainable
3. To create a complete solution for interface frontend ecosystem
